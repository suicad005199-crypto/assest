# AI Sports Prediction System

## Overview

This project is an AI-driven sports prediction system for football and basketball.

It uses multi-agent reasoning to generate:

- Win probability
- Predicted score
- Expected Value (EV)
- Risk evaluation
- Betting recommendation

The system is designed for long-term profitability, not perfect prediction accuracy.

---

## Core Objective

The goal is NOT to predict every match correctly.

The goal is to:

- Identify positive Expected Value (EV)
- Beat market odds over time
- Maintain stable ROI

---

## Supported Sports

- Football (Soccer)
- Basketball (NBA / EuroLeague)

---

## Output Goals

Each prediction must include:

- Win / Draw / Loss probability
- Predicted score
- Confidence level
- Risk level
- Expected Value (EV)
- Recommendation (BET / NO BET)

---

## Key Metrics

### Football

- ELO Rating
- xG / xGA
- Home/Away performance
- Injuries
- Tactical matchup
- Recent form

### Basketball

- ELO Rating
- Offensive Rating
- Defensive Rating
- Pace
- TS% / eFG%
- Injuries
- Rest days

---

## Expected Value Formula

EV = (Probability × Odds) - 1

A bet is valid only when:

EV > 0
